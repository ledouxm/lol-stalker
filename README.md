# LoL friendlist ranking stalker

![LoL friendlist ranking stalker](Screenshot.png)

## Installation

Clone this repo and install all dependencies  
`yarn` or `npm install`

## First start

`yarn init:db`

## Development

`yarn dev` or `npm run dev`

## Build

`yarn build` or `npm run build`

## Publish

`yarn dist` or `npm run dist`
